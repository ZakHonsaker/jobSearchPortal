# jobSearchPortal

# installation

  1. clone repo https://github.com/merveecolakoglu/jobSearchPortal.git
  2. create a virtual environment and active
  3. cd 'jobSearch'
  4. pip install -r requirements.txt
  5. python manage.py runserver

  - Django, Postgres
